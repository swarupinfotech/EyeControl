## **Overview**
Welcome to **EyeControl**, an innovative eye-tracking system that enables users to interact with digital devices using only their eyes. This project demonstrates how eye movements can replace traditional input methods like a mouse or keyboard, offering a seamless, hands-free experience.

**EyeControl** is designed to enhance accessibility, improve user interaction, and create a more natural interface for both personal and professional use. It leverages advanced computer vision algorithms and machine learning techniques to track and respond to eye movements in real time.

---

## **Key Features** ‚ú®
- **Hands-free navigation**: Navigate through your device with only your eyes.
- **High precision**: Accurately tracks eye movements for smooth control.
- **Customizable settings**: Adjust sensitivity, gaze duration, and control methods to suit your needs.
- **Cross-platform support**: Compatible with Windows, Linux, and macOS.
- **Accessibility**: Provides a solution for individuals with limited mobility or disabilities.

## **Installation Instructions** üöÄ

### 1. Clone the Repository
First, clone the repository to your local machine:
```bash
git clone https://github.com/swarupinfotech/EyeControl.git

```
## 2. Install Dependencies
Make sure you have Python 3.x installed. Then, install the necessary dependencies:
```bash
cd EyeControl
pip install -r requirements.txt

```
## 3. Run the Application
To start the application, simply run the following command:

```bash
python eye_tracking_app.py
```

## 4. Set Up Eye-Tracking Hardware
Follow the hardware setup instructions provided in the repository to connect and calibrate your eye-tracking device.
